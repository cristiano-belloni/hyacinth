Hyacinth

Host / Plugin interface for Web Audio Applications.

What is Hyacinth

The Hyacinth project is an interface specification for plugin-based Web audio applications, plus an open host implementation and an open plugin repository. Plugins are modular, independent pieces of code dynamically loaded and interconnected by an Host application. the Hyacinth specification describes how plugins and hosts interact.

Name

Hyacinth comes from T.S.Eliot's The Waste Land.

Specification

The Hyacinth specification is an open document which describes how Hyacinth-compliant hosts and plugins interact. You can find the latest version of the specification in the docs section of this site, or contribute to it forking the relative Github repository and sending a pull request.

Hyacinth Host

Hyacinth Host (formerly known as KievII Host) is a web application which implements the Hyacint specification. It can dynamically load Hyacinth plugins and gives the user a GUI to manage and interconnect them.

Hyacinth Plugins

Hyacinth Plugins (formerly known as KievII Plugins) is an open collection of audio plugins, stored in a GitHub repository. The plugins implement the Hyacinth specification and will work with any Hyacinth-compliant host.
